This project is a Python-based CRUD (Create, Read, Update, Delete) application that interacts with a MySQL database to manage user data. It allows users to perform database operations through a command-line interface.

Key Features:
Insert Data: Adds a new user with a name, age, and city to the database.
Update Data: Modifies existing user details based on the user ID.
Select Data: Retrieves and displays all user records in a formatted table using the Tabulate library.
Delete Data: Removes a user from the database by their ID.
Menu-Driven Interface: Provides an interactive menu for easy navigation and operations.
Technologies Used:
Python for backend logic
MySQL for database management
Tabulate for displaying data in a readable table format
This project demonstrates how to integrate Python with MySQL to perform basic database operations efficiently.
